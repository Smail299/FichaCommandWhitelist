# FichaCommandWhitelist
Плагин, заменяющий стандартные сообщения о недоступности команд. Если у игрока недостаточно прав, он видит, какая именно привилегия нужна для выполнения команды. Работает как дополнение к CommandWhitelist.

# Подробное описание:
Этот плагин является дополнением к CommandWhitelist. С этим плагином при вводе команды, требующей привилегии выше вашей, в чат будет писаться сообщение о том, с какой именно привилегией будет доступна данная команда. Также если в конфиге не указать привилегию, то при вводе команды этой привилегии игроку не будет приходить сообщение о том, что она доступна с этой привилегии, вместо этого игроку будет приходить дефолтное сообщение которое вы указали в конфиге CommandWhitelist. Идея была взята с сервера HolyWorld Lite.
